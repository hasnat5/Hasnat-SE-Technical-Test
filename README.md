## Run Locally

Clone the project

```bash
  git clone https://github.com/hasnat5/Hasnat-SE-Technical-Test.git
```



### Soal no 1
Go to the project directory

```bash
  cd test-1
```

Run the code

```bash
  node tes1.js
```




### Soal no 2
Go to the project directory

```bash
  cd test-2
```

Run the code

```bash
  node tes2.js
```




### Soal no 3 Frontend
Go to the project directory

```bash
  cd test-3
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
